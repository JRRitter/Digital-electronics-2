Generating analog signal through a 8-bit R2R ladder with keyboard control and LCD display.

This program immitates a signal generator. It is able to create 3 different waveforms. The whole program is controlled by a 4x3 keyboard and an LCD display is used. Useful information is sent to a remote device via uart. LCD and uart control are managed with libraries created by PEter Fleury.
